# twitter-simulator-v2-static

无需 npm，直接双击 `index.html` 即可运行。

## 已实现
- 编辑/查看模式切换（查看模式隐藏编辑入口）。
- 账户管理：新增、修改昵称/ID、头像上传、删除、设置浏览账户。
- 发帖：可选发送账户、可手改时间、可勾选翻译块。
- 回复：可选发送账户。
- 推文/回复翻译块：可显示/隐藏、可编辑翻译来源与译文。
- 搜索：按作者/ID/正文/翻译/时间过滤当前时间线。
- 多语言 UI：中文/日文切换。
- 持久化：localStorage + IndexedDB（头像/图片）刷新后保留。
- 编辑模式导入/导出 JSON；导出 HTML 快照。

## 运行
1. 打开 `twitter-simulator-v2-static/index.html`。
2. 首次加载后所有数据写入浏览器本地存储。

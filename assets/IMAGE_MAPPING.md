# Default Image Mapping

Put your default images in the folders below using exactly these filenames.

## 1) Account avatars (used by default tweets/accounts)

Folder: `assets/default/avatars/`

- `avatar-yachiyo.jpg` -> ヤチヨ / `@yachi8000`
- `avatar-tsukuyomi-news.jpg` -> 月読ニュース / `@tsukuyomi_news`
- `avatar-irokaguforever.jpg` -> 彩輝激推し / `@irokaguforever`
- `avatar-kaguya.jpg` -> かぐや / `@kaguya`
- `avatar-nihohecy.jpg` -> 色々 / `@nihoheYCY` (optional)

## 2) Tweet media images (used by default tweets)

Folder: `assets/default/tweets/`

- `tweet-1-main.jpg` -> tweet `id: 1`
- `tweet-2-main.jpg` -> tweet `id: 2`
- `tweet-3-main.jpg` -> tweet `id: 3` (optional, currently no media)

## 3) Optional UI avatars (sidebar recommendation area)

Folder: `assets/default/ui/`

- `follow-1.jpg` -> 推荐关注第 1 个账号
- `follow-2.jpg` -> 推荐关注第 2 个账号
- `follow-3.jpg` -> 推荐关注第 3 个账号
- `sidebar-viewer.jpg` -> 左下角当前浏览账号头像（可选）
- `compose-viewer.jpg` -> 顶部发帖框头像（可选）

## Recommended format

- Avatar: square (at least `400x400`), JPG/PNG/WebP
- Tweet media: ratio close to `16:9` (at least `1280x720`)
- Use lowercase + hyphen names to avoid path/case issues
